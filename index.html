Projects

PacmanGo

PacmanGo is a spin-off on the original Pacman game, in which ghost can pass through the walls! Relevant parts of the project are the controller conversion from keyboard to mouse and the map implementation. The map is defined as a 2D int ("bool") array of 0's and 1's, signifying a wall or empty space. When a Map object is initialized, it converts the array into an array of Polygons and everytime it finds a wall segment, it recursively finds the whole wall which contains that segment. Then, it is turned into a polygon, and the values of the found wall in the array are all set to 0. The collision detection, on the other hand, checks if the Pacman bounding box intersects with any of the polygons of the map.
